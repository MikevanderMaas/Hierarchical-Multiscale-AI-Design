# Hierarchical-Multiscale-AI-Design
In this repo there are two versions of Multiscale AI design as originally ideated by Weber. A version with and without hierarchical planning is added as they were part of a research, link to come later.

The multi-scale versions seperate the planning entirely from execution, using the c/h_MultiScale class to orchestrate the actual functionality.
The hierarchical version has an extra layer of planners, based on the results of the research these shorten the time it takes for the desgin to create a plan.
